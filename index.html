Studiehjelp

Studiehjelp is a website made to aid teachers in their day to day lives, by saving time they usually spend on the adaption of education. It does this by automatically adapting texts to a students age, reading difficulty and language, as well as any extra adaptations, by using the ChatGPT4 API.

  • React
  • Express
  • ChatGPT API
  • Redis
  • MongoDB
Check out the website!

NTNUPPL

NTNU pingpong league is a website made for the final exam of the course full stack web development. It is a website where you can see the current standings, the schedule and the results of the matches, as well as play matches locally or through a websocket.

  • React
  • Express
  • MongoDB
  • Refreshtoken
  • Websocket
Check out the website!

CapMap

CapMap is a collaborative effort between two web developers and three interaction designers. It features an interactive map for viewing a rooms occupancy, temperature and noise level, as well as a histogram to see previous occupancy. Occupancy is monitored by a raspberry pi and two ultrasonic sensors placed in a doorway. Communication is handled by MQTT.

  • Python
  • Bash
  • React
  • Express
  • MQTT
  • Redis
