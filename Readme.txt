This project will have 10 scenes about learning how to get into music production for video games and learn how to produce.
We will also create one square each with keyframes.





What the project does:
This project will teach music production in a way that is centered around music in videogames. We will go over a few different techniques and some software, we will also show a few examples of how music is produced, and then used in videogames.

Why the project is useful:
This project is useful because there isn't many easily accessible resources for music production around videogames.

How users can get started with the project:
To get started with the project, just follow the steps on our website. 

Where users can get help with your project:
We will add an email address at the footer of our website

Which technology is used:
The technology used is just HTML, CSS and Javascript, also some drawing programs to prototype, such as figma and adobe illustrator. And of course visual studio code. We may also use some websites such as github, codepen, and clippy.
